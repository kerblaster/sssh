Smash Set Helper
Instructions
Concise instructions for navigating this app
Home
  • Logo - Pressing the logo on the top left of every page will return you back to the home screen
  • Games include: Smash Bros, Melee, Brawl, Project M, and Smash 4.
  • Select a smash title from the home screen to begin
After selecting a title...
  1. Ruleset available in text of the selected smash title
  2. There are two competitive game modes:
    • Singles, which is a one versus one raw experience and
    • Doubles, which you pair with a friend and go against another pair
  3. Select a what the series is out of:
    • Best of 1 is a winner takes all
    • Best of 3 (first to two wins) is the standard competitive set
    • Best of 5 (first to three wins) is the standard for top 8 play in tournaments or money matches
  4. Choosing characters:
    • Players may choose from the cast below
    • The red border represents the opponent's character after losing
    • The green border represents your friend's character in doubles
  5. RPS port or stage strike:
    • Randomizes which player wins RPS if would have elected to do so
    • Choosing port enables gives players the right to choose what port (1-4) to insert their controller.
    • The stage strike option grants the players the last ban out of two stages.
    • If you are chosen randomly and don't want first pick, select 'Choose Port.'
    • Player/Team who has port priority can always opt to pick Port 2, 3 or 4 (not necessarily port 1).
  6. Neutral stage pick:
    • This is the stage striking phase.
    • Note that some maps will be initially banned because they are not neutral stages.
    • Ban a stage if it is your turn to ban.
    • It does not matter who picks the last stage because there will only be one available in the end.
    • If both parties agree to a stage not in the legal stagelist, select 'Mutual Agreement'
  7. During an active game:
    • Play out your game with the following basic rules stated
    • For more information, visit the game's ruleset in the menu
    • Select the left character if left won.
    • Select the right character if right won.
    • There is a timer active which displays when set is complete
  8. Post-game stage picks:
    • If the game and best-of series ruleset supports it, the winner may ban stage(s). The loser, no matter what, picks a stage.
    • Counterpick stages are now available.
    • If you are chosen randomly and don't want first pick, select 'Choose Port.'
    • Note: DSR states players may not choose a stage they already won on in the same set, unless mutually agreed to.
  9. Post-game character selection:
    • Winner may opt to change characters
    • Loser then may opt to change characters
  10. Set completion:
    • Coin indicates winner of the set
    • Green border indicates winner of the game
    • Red border indicates loser of the game
    • Games and sets that require only 1 ban post-game can tap a stage (after game 1) to see what they have banned during the set
Menu
  • Instructions: You are here!
  • Donate: If you support this app, the button is located here for a paypal donation link
  • Hand warmers: 30 sec countdown
  • Set DQ: 10 min countdown and 5 min countdown
  • Rock Paper Scissors: Randomizes left or right with equal chance
